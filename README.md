## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** JavaScript
- **Package Manager:** Npm

### Installation

Build Notes from the source and install dependencies:

1. **Clone the repository:**
```bash
git clone https://github.com/astrax-indesire/Notes
```

2. **Navigate to the project directory:**
```bash
cd Notes
```

3. **Install the dependencies:**

Using **npm**:
```bash
npm install
```

### Usage

Run the project with:

Using **npm**:
```bash
npm start
```

### Testing

Notes uses the **itest_framework** test framework. Run the test suite with:

Using **npm**:
```bash
npm test
```
